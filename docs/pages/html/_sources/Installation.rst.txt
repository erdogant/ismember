Pip install
################################

.. code-block:: console

    # Install from Pypi:
    pip install ismember

    # Install directly from github
    pip install git+https://github.com/erdogant/ismember


Uninstalling
################

.. code-block:: console

   # Remove the ismember environment
   pip uninstall ismember
